# Modded-Prusa-MK2.6s

My MK2.5S mods done to it which I call it now an Prusa MK2.6s

# Mods:
- Bear Frame
- Dual LEDs strips
- Mk3 Extruder
- Improved Filament Sensor Setup/housing
- Stepper Driver Board (12v) with TMC2209
- MMU3 Ultimulti Version (https://www.printables.com/model/537888-mmu3-ultimulti-printable-parts)
